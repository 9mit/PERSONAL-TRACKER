# 📊 Personal Tracker Pro

A high-speed, lightweight task and annotation tracker designed to log detailed work sessions, display real-time dashboard analytics, and automatically manage database storage via a daily user-checkout system.

---

# 📘 Personal Tracker Pro — User Manual

# 🏠 Introduction

Welcome to **Personal Tracker Pro**.

This platform is designed to help users track daily work activities, monitor productivity, submit task information, and generate reports in real time.

The system is simple, lightweight, and easy to use for both employees and managers.

---

# 🎯 Purpose of the Website

The website helps users:

* Submit daily work information
* Monitor productivity
* Track performance
* View analytics dashboards
* Export reports
* Maintain transparent work records

---

# 🔑 Getting Started

## Step 1: Open the Website

Open the website link in your browser.

Recommended browsers:

* Google Chrome
* Microsoft Edge
* Firefox

---

## Step 2: Login to Your Account

Enter your:

* Username or Email
* Password

Click on the **Login** button.

After successful login, you will be redirected to the dashboard.

---

# 🖥️ Dashboard Overview

The dashboard is the main screen of the application.

Here you can:

* Submit work entries
* View today’s progress
* Check analytics
* Monitor shift activity
* Download reports
* Track performance summaries

---

# 📝 How to Submit Daily Work

## Step 1: Open the Work Submission Section

From the dashboard:

* Click on **Submit Work** or **Add Entry**.

---

## Step 2: Enter Required Information

Fill in the required details such as:

* Task name
* Number of questions/tasks completed
* Shift or slot
* Additional comments (optional)

Make sure all required fields are filled correctly.

---

## Step 3: Submit the Entry

Click the **Submit** button.

Once submitted:

* The data is saved automatically.
* Your dashboard updates in real time.
* Managers can instantly view the update.

---

# ⚡ Real-Time Updates

The website supports live updates.

This means:

* New submissions appear instantly.
* Dashboard statistics refresh automatically.
* No manual page refresh is needed.

If multiple users are using the system, everyone sees updates live.

---

# 📊 Viewing Analytics

## Analytics Section

Open the **Analytics** or **Reports** section from the dashboard.

Here you can view:

* Daily productivity
* Work summaries
* Shift performance
* Submission counts
* Activity trends

---

# 📅 Shift/Slot Tracking

The system automatically organizes entries into shifts or work slots.

Users can:

* View current shift records
* Check shift-wise productivity
* Monitor work completion status

# 📂 Exporting Excel Reports

## Step 1: Open Reports Section

* Reports
* Export
* Download Data

(depending on the interface design)

---

## Step 2: Select Report Type

Choose the report you want:

# ⚠️ Important Guidelines

## Do:

* Submit accurate work data.
* Check entries before submission.
* Download reports regularly if needed.
* Logout after completing work.

## Don’t:

* Share login credentials.
* Submit duplicate entries.
* Refresh repeatedly during submission.

---

# 🚪 Logout Procedure

After completing your work:

1. Click on the profile/account icon.
2. Select **Logout**.
3. Close the browser if required.

This keeps your account secure.

# 🛠️ Troubleshooting Guide

## Problem: Unable to Login

Possible reasons:

* Incorrect password
* Invalid username
* Internet issue

Solution:

* Recheck credentials
* Reset password if available
* Contact administrator

---

## Problem: Submission Not Saving

Possible reasons:

* Missing required fields
* Network interruption

Solution:

* Fill all required fields
* Refresh and retry

---

## Problem: Dashboard Not Updating

Solution:

* Check internet connection
* Refresh browser once
* Re-login if needed
